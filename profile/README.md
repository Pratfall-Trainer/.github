**Installation Guide**  
1. Download the latest archive.  
2. Extract to a dedicated folder.  
3. Run `PratfallTrainer.exe` as administrator.  
4. Launch Pratfall.  
5. Press INSERT to open the trainer interface.  
6. Activate features via hotkeys or menu.  

Re-attach after game restarts or major level loads.

**Safety & Risk Information**  
- **High risk of save file corruption**: Extreme score multipliers, physics edits, or progression changes can permanently damage your save. Always create multiple backups before every session.  
- Heavy physics modifications may cause temporary lag, clipping, or unstable ragdoll behavior.  
- Rapid changes during autosave cycles frequently result in lost progress or broken level states.  
- The trainer process commonly triggers antivirus detections due to memory injection.  
- Test all features in a new run first. No recovery possible for corrupted saves.
